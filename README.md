SETUP/NOTES:
1. This API was developed using C# 10, .NET 8.0, and Visual Studio 2022. Make sure that your C#, .NET, and Visual Studio are all running on those versions.
2. Git Clone the this repository directly at the Visual Studio IDE as it will automatically download/install the necessary NuGet Packages for you.
3. Make sure that you run the API in the 44313 PORT by chaning the run environment from https/http to IIS Express at the menu bar of your Visual Studio IDE.

NOTE:
- If you cannot run the API or your Visual Studio in IIS or 44313 PORT. You can just manually change the port number of the API endpoint in the environment.ts
  of the Angular project in Visual Studio Code.
